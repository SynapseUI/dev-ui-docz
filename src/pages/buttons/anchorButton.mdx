## Props

| Property        | Type      | Required | Description             |
| :-------        | :---      | :------- | :----------             |
| className       | `string`  | false    | className of the Button |
| primary         | `boolean` | false    | Set Primary style       |
| secondary       | `boolean` | false    | Set Secondary style     |
| tertiary        | `boolean` | false    | Set Tertiary style      |
| remove          | `boolean` | false    | Set Remove style        |
| text            | `string`  | true     | Text within the anchor tag        |
| onClick         | `function`| false    | The onclick function that gets trigged when clicked       |
| fontSize        | `string`  | false    | Font size of the text        |
| disabled        | `boolean` | false    | Applies a disabled property, onclick will be disabled        |
| underLine       | `boolean` | false    | Applies a underline        |
| color           | `string`  | false    | Hex code of the text font color        |
| hoverColor      | `string`  | false    | Hex code of the hover text font color        |
| activeColor     | `string`  | false    | Hex code of the active text font color        |
| backgroundColor | `string`  | false    | Hex code of the background color        |
