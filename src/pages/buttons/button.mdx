## Props

| Property   | Type      | Required | Description                                        |
| :-------   | :---      | :------- | :----------                                        |
| className  | `string`  | false    | className of the Button                            |
| primary    | `boolean` | false    | Set Primary style                                  |
| secondary  | `boolean` | false    | Set Secondary style                                |
| tertiary   | `boolean` | false    | Set Tertiary style                                 |
| remove     | `boolean` | false    | Set Remove style                                   |
| fullWidth  | `boolean` | false    | Set Full Width style                               |
| isDisabled | `boolean` | false    | Set isDisabled style, will prevent on click action |
| isLoading  | `boolean` | false    | Set isLoading style, will prevent on click action  |
