# SwitchButton

import { PropsTable } from 'docz';
import SwitchButtonContainer from './switchButton.state';

## Basic

<SwitchButtonContainer />

```jsx
  <SwitchButton
    value={this.state.switch}
    propName="switch"
    onChange={(e, value) => this.setState({ switch: value })}
  />
```

## Props

| Property  | Type       | Required | Description                   |
| :-------  | :---       | :------- | :----------                   |
| value     | `bool`     | true     | Value of SwitchButton         |
| className | `string`   | false    | className of the SwitchButton |
| onChange  | `function` | true     | Function triggered on toggle  |
| propName  | `string`   | false    | Name of the state key         |
