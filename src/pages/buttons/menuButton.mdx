# Menu Button

## Basic


<MenuButtons />

```jsx
<MenuButtons
  data={
    [
      { key: 'one', text: 'One', onClick: () => alert('one') },
      { key: 'two', text: 'Two', onClick: () => alert('two') },
    ]
  }
/>
```

## Props

| Property  | Type     | Required | Description                                    |
| :-------  | :---     | :------- | :----------                                    |
| data      | `array`  | true     | List of menu items, needs "text" and "onClick" |
| className | `string` | false    | Optional class name                            |
