# FloatingActionButton


import FABContainer from './FABContainer.state';

## Basic
<FABContainer />

```jsx
  <FloatingActionButton
    hoverText="Example Floating action button"
    panelHeader="Example Floating action button"
    panelContent={/** some component */}

    data={floatingActionButtonData}
  />
```

## Data

```jsx
const floatingActionButtonData = {
  Test1: {
    onClick: () => console.log('hi'),
    text: 'test 1',
    icon: <SvgIcons.crypto_wallet size="24px" />
  },
  Test2: {
    onClick: () => console.log('hi'),
    text: 'test 2',
    icon: <SvgIcons.deposit_account size="24px" />
  },
  Test3: {
    onClick: () => console.log('hi'),
    text: 'test 3',
    icon: <SvgIcons.payment size="24px" />
  },
};

```

## Props

| Property     | Type                    | Required | Description                                               |
| :-------     | :---                    | :------- | :----------                                               |
| hoverText    | `bool`                  | true     | Defines when the modal is visible                         |
| panelHeader  | `string`                | true     | Header Text                                               |
| panelContent | `componet`/`DOMelement` | true     | Content of the Panel                                      |
| data         | `object`                | false    | Panel can open a list of items before the content is show |
