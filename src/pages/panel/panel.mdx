# Panel


import PanelContainer from './PanelContainer.state';

## Basic
<PanelContainer />

```jsx
  <Panel
    title="Panel isOpen value is outside the component"
    isPanelOpen={this.state.panelIsOpen}
    onPanelClose={() => this.setState({ panelIsOpen: false })}
  >
    <p>Useful if you need to open without a button, i.e. componentDidMount</p>
  </Panel>

  <Button secondary small onClick={() => this.setState({ panelIsOpen: true })}>
    Open Panel
  </Button>
```

## Props

| Property     | Type                    | Required | Description                       |
| :-------     | :---                    | :------- | :----------                       |
| title        | `string`                | false    | Header text |
| onPanelClose | `function`              | true     | Function that hides the panel     |
| openPanelButton| `component`           | true     | Component that opens the panel when clicked |
| onClickBeforeOpen | `function`         | false    | Function executed before panel opens     |
| isPanelOpen  | `bool`                  | true     | Defines when the modal is visible |
| downloadButtonData| `object`           | false    | Data for download button, which downloads a file on button click. {`fileDownload`,`icon`} |
| emailButtonData  | `object`            | false    | Object with `recipient`, `subject`, and `body` strings, which routes user to an email when clicked |
