## Props

| Property         | Type      | Required | Description                                                      |
| :-------         | :---      | :------- | :----------                                                      |
| status           | `string`  | true     | Value of the tag                                                 |
| mini             | `boolean` | false    | If mini prop and icon exist than it returns the icon             |
| trim             | `boolean` | false    | If trim prop and trimText exist than it returns the trimmed text |
| className        | `string`  | false    | className of input                                               |
| customStatusData | `string`  | false    | Custom style and text                                            |
