# CountryPhoneInput


import { CountryPhoneInput } from 'synapsefi-dev-ui';
import CountryPhoneInputContainer from './CountryPhoneInput.state';

## Basic
<CountryPhoneInputContainer />

```jsx
  <CountryPhoneInput
    value={this.state.countryPhoneNumber}
    propName="countryPhoneNumber"
    onChange={(e, value) => this.setState({ countryPhoneNumber: value })}
    options={[
      { key: 'USA', text: 'USA' },
      { key: 'GBR', text: 'GBR' },
      { key: 'CAN', text: 'CAN' },
    ]}
    getCallingCode={getCountryNumber}
  />

  // getCountryNumber function
  const getCountryNumber = code => {
    const list = {
      USA: 1,
      GBR: 44,
      CAN: 1,
    };

    return list[code] || 1;
  };
```

## Props

| Property    | Type             | Required | Description |
| :-------    | :---             | :------- | :---------- |
| value       | `string`/`array` | true     | Value of the Input |
| propName    | `string`         | false    | Property name |
| className   | `string`         | false    | className of input |
| label       | `string`         | false    | Label of input |
| onChange    | `function`       | false    | Function fired when input's `onChange` event triggers. Arguments: (e, value, propName).|
| onFocus     | `function`       | false    | Function fired when input's `onFocus` event triggers.|
| placeholder | `string`         | false    | Placeholder of input |
| disabled    | `boolean`        | false    | Disabled of input |
